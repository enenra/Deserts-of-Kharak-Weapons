> A pack of weapons based on the game Deserts of Kharak, made with WeaponCore. There currently is only one turret in this mod, but I'm looking to eventually expand it with more weapons found in the game. Also check out my [Coalition LAV](https://steamcommunity.com/sharedfiles/filedetails/?id=3690603398) build that I used for the screenshots!


![img](https://raw.githubusercontent.com/enenra/Deserts-of-Kharak-Weapons/refs/heads/main/OriginalContent/Promo/Header%20Features.png)
This mod currently contains one turret:

* Manaan Dual Autocannon (SG / LG)


![](https://raw.githubusercontent.com/enenra/Deserts-of-Kharak-Weapons/refs/heads/main/OriginalContent/Promo/Header%20Notes.png)

* This turret will need more balancing - I've only done some preliminary balancing with it so far.
* I will probably eventually be making more of the Deserts of Kharak turrets but making these is more of a matter of if I feel like it, and do not have priority over my other modding projects.
